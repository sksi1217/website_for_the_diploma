# website_for_the_diploma

АИС «Успеваемость школьников» — веб-приложение для дипломного проекта.

## Стек

- Python 3, Flask
- SQLite
- HTML, CSS, JavaScript

## Запуск

```bash
pip install -r requirements.txt
python main.py
```

Сайт откроется по адресу: http://localhost:5000

## Возможности

- Учёт классов, учеников, предметов и оценок
- Поиск и фильтрация
- Отчёты и экспорт в CSV
- Демо-данные для тестирования
